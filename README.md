# CCAPDEV-MCO
To start the app, you will need 2 terminals.
For the first terminal,
1. do "cd taftics/server"
2. run "node server.js"
For the second terminal
1. do "cd taftics"
2. run "npm run dev"

After connecting to the MongoDB, open your mongodb compass and refresh and go to the taftics collection.
For each collection (comments, establishments, reviews, and users), import the data from the imports folder and select the json file with the same name.
